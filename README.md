# Students-Registration
A responsive Student Registration Form built with HTML, CSS, and JavaScript. Allows students to register with personal details, upload profile images, and view all records on a separate page. Data is stored locally using localStorage with options to delete entries.

Description:
This is a Student Registration System built using HTML, CSS, and JavaScript. The project provides a user-friendly front page and a registration form where students can input their details, including:

Full Name

Email

Phone Number

Password & Confirm Password

Date of Birth

Gender

Skills (HTML, CSS, JS, React, Node.js)

Education

Address

Profile Image Upload

All student data is stored locally using localStorage, and users can view their registered students on a separate Students Page. Each student card displays the details along with a Delete button to remove the record.

Features:

Responsive design compatible with mobile and desktop screens.

Real-time form validation (email format, phone number, password matching, required fields).

Image preview functionality before upload.

Smooth animations for headings and buttons.

Simple, interactive UI using CSS and JavaScript.

Local storage integration to persist student data without a backend.

Technologies Used:

HTML5

CSS3 (Flexbox, Animations, Media Queries)

JavaScript (DOM manipulation, Event handling, localStorage)

Usage:

Open frontpage.html to access the landing page.

Click Register to go to the registration form.

Fill out the student information and submit.

View registered students in students.html.

Use the Delete button to remove any student record.

Potential Improvements:

Integrate a backend (Node.js, Express, or Firebase) to store data persistently.

Add search and filter functionality for student records.

Improve accessibility and UI with modern frameworks like Bootstrap or Tailwind CSS.
